# 1. GitHub Wiki

## Overview

Today, I learned that GitHub Wiki is a built-in documentation feature provided by GitHub that allows teams to maintain project-related information in an organized manner. Instead of storing all documentation inside the repository, a Wiki provides dedicated pages for maintaining technical documents.

## Key Learnings

- Understood the purpose of GitHub Wiki.
- Learned how project documentation is organized using multiple Wiki pages.
- Learned that a Wiki can contain installation guides, project overviews, API documentation, contribution guidelines, and troubleshooting information.
- Understood that GitHub Wiki helps maintain project knowledge in a centralized location.
- Learned the importance of keeping documentation updated throughout the software development lifecycle.
